What is FileLock?

File lock is software that uses our special double encryption technique. This software makes it extremely hard for those potent USB thieves to get into your files. File lock also has a backup feature in case your USB gets lost or stolen. FileLock does so by automatically syncing your files with your Microsoft OneDrive account. However, it will do so if you have a Microsoft OneDrive account. If you do not have one it will not disrupt the performance of the software. if you have one of course.
